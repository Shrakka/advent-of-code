Tc = Time of charging 
Tt = Time of travel 
V = Speed of travel
D = distance of travel


Sur le premier cas d'exemple, on a les valeurs : 
- Temps de course : 7
- Distance minimum : 9

On sait que ( Tc + Tt = 7 ) 
On sait que ( Tc = V )


On cherche :
D             > 9
V  * Tt       > 9
Tc * Tt       > 9
Tc * (7 - Tc) > 9

7Tc - Tc^2 - 9 > 0

ie, -Tc^2 + 7Tc - 9 > 0

ie -X^2 + 7X - 9 > 0

ie, -X^2 + TIME * X - DISTANCE > 0

a = -1
b = time
c = - distance

et on veut :
x > 0
x < TIME
